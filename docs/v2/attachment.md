# Attachment

## オブジェクト

| name    | type    | description         |
| :---    | :---    | :---                |
| name    | string  | ファイル名            |
| url     | string  | ファイルのURL         |
| height  | ?number | 画像の高さ            |
| weight  | ?number | 画像の横の長さ         |
| content_type | ?string | ファイルのタイプ名 |
